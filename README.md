# 💬 Sentiment Analysis App

**Built by Sujitha | AI/ML Portfolio Project**

## 📌 About
NLP-based sentiment analyzer that classifies 
Amazon product reviews as Positive or Negative.

## 🎯 Results
- ✅ Accuracy: 92%
- ✅ Dataset: 568,000 Amazon Reviews
- ✅ Live Predictions via Streamlit UI

## 🛠️ Tech Stack
Python | NLTK | TF-IDF | Logistic Regression | Streamlit

## 🚀 How to Run
git clone https://github.com/sujithakannan25/sentiment-analysis.git
pip install -r requirements.txt
streamlit run app.py
